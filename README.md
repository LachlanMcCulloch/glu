# glu

A simple TypeScript CLI tool built with Commander.

## Installation

Clone the repository and install dependencies:

```bash
npm install
```

To install globally:

```bash
npm install -g .
```

## Usage

After building:

```bash
npm run build
node dist/index.js hello
```

Or if installed globally:

```bash
glu hello
```

## Development

- Build: `npm run build`
- Start: `npm start`

## Commands

- `hello`: Prints a hello message
