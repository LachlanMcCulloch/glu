# GLU – Git Layer Utility

**GLU** is a command-line tool that perfects the Git stacking experience. It
helps you manage layered branches, rebase with confidence, and ship clean,
incremental changes — without getting lost in merge hell.

> Think of it as “git stack” — but composable, scriptable, and stress-free.

> **Note**
> This is a WIP and not ready for prod use yet.

---

## ✨ Planned Features

- 🔁 **Layered branching**: Create, view, and manage stacks of commits as
  branches.
- 📌 **Auto-rebase**: Rebase stacks incrementally — one branch at a time.
- 🧠 **Minimal context switching**: Always know where you are in the stack.
